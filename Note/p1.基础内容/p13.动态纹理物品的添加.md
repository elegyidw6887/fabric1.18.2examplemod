## 1.物品对象的创建与注册

## 2.物品模型以及动态纹理的添加
- 动态纹理物品所需要的模型json文件与普通物品相同
- 纹理文件有所不同
- 动态纹理需要一个由多个纹理组成的png格式文件以及一个.mcmeta格式文件
```mcmeta
{
  "animation": {
    "frametime": 16,
    "interpolate": true,
    "frames": [
        0,
        1,
        2,
        3,
        4,
        5
    ]
  }
}
```
- mcmeta格式文件如上所示
- “animation”表示其为动态纹理
- “frametime”表示其png文件中多个部分之间的切换时间帧
- “interpolate”表示这些部分之间不会直接切换而是进行过度
- “frames”表示你png文件中共有几个部分
